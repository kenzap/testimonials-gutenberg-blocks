== Description ==

A beautiful and easy customizable set of Gutenberg blocks to create Testimonials table section for the new editor. Easily adjust the following parameters:

Set background color
Change featured table layout style
Adjust images, fonts, texts, price and currencies
Wide and narrow layout support
Supports 5 different design layouts

== Installation ==

This plugin can be installed directly from your site.

1. Log in and navigate to _Plugins &rarr; Add New_.
2. Type "Kenzap Testimonials” into the Search and hit Enter.
3. Locate the Kenzap Testimonials plugin in the list of search results and click **Install Now**.
4. Once installed, click the Activate link.
5. Go to Pages > Add New > Find Testimonials block.
6. Adjust Container > Max width setting if elements are not displayed properly. 


It can also be installed manually.

1. Download the Kenzap Testimonials plugin from WordPress.org.
2. Unzip the package and move to your plugins directory.
3. Log into WordPress and navigate to the Plugins screen.
4. Once installed, click the Activate link.
5. Go to Pages > Add New > Find Kenzap Testimonials blocks.
6. Adjust Container > Max width setting if elements are not displayed properly. 


== Frequently Asked Questions ==

1. What is this plugin for?
To extend your current theme with Testimonials section

2. Why columns look narrow?
Depending on your front page container layout type you may need to reduce Max width property in Inspector container
